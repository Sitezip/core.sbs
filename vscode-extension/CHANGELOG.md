# Change Log

All notable changes to the "core.js Snippets" extension will be documented in this file.

## [1.0.0] - 2026-01-28

### Added
- Initial release of core.js snippets extension
- HTML snippets for pockets, templates, and clones
- JavaScript snippets for lifecycle hooks and API calls
- Template directive snippets (data, rec, aug)
- Helper function snippets
- Validation and scrubbing snippets
- Complete example snippets
- Form handling snippets

### Snippet Categories
- **Pockets**: `cpk`, `cpkd`, `cpkt`, `cpkid`, `cpknr`
- **Templates**: `ctpl`, `ctpld`, `ctplc`, `ctplf`
- **Clones**: `ccl`, `cclr`, `cclnest`, `cclaug`
- **Directives**: `cdata`, `crec`, `caug`, `cattr`, `ccloner`, `cformat`
- **Hooks**: `cudinit`, `cudsoc`, `cudeoc`, `cudpre`, `cudpost`, `cudprepaint`, `cudpostpaint`
- **API Calls**: `cbeget`, `cbetpl`, `cbepost`, `cpkinit`, `ccrset`, `ccrget`
- **Helpers**: `chfdig`, `chfdate`, `chfhydrate`, `chfformat`, `chfparse`
- **Validation**: `csvscrub`, `csvformat`
- **Anchors**: `caget`, `catpl`, `capk`
- **Examples**: `cexample`, `cform`
